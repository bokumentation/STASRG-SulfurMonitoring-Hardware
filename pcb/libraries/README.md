**Custom Board.kicad_sym** contains all symbols with assigned footprints


This folder are for Footprint and Symbols. You should add them into one single folder with this name convention: YYMMDD_NAME_REV

`.pretty` are the footprint.
`.kicad_sym` are the symbols.

To import this library just copy the `.pretty` and `.kicad.sym` into your project folder.
